# Wellness

Full-stack wellness / habit tracking application (personal learning project 2026)

## Current status
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Running locally with `npm run dev`

## Planned features
- User authentication
- Daily mood / energy logging
- Habit tracker with streaks
- Simple dashboard + charts
- PostgreSQL database (Prisma + Neon.tech)
- Deployment to Vercel

## How to run locally

```bash
git clone https://github.com/faria-A7/wellness.git
cd wellness
npm install
npm run dev
